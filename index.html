Projects

Non-Contact Handwriting Recognition System

  • This project will establish a non-contact handwriting recognition system, which uses acoustic pulse signal and cheap acoustic sensor for input human-computer interaction. When the hand moves on the desktop, the ultrasonic receiving module captures the sound signal of the transmitting module, and then restores the handwriting track to recognize characters. In addition, the tablet is implemented as a real-time recognition system, and a comprehensive experiment will be carried out to verify the effectiveness and robustness of the system. Finally, a contactless handwriting recognition system which can be used in practical application is established.
